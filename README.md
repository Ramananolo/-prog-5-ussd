# 📱 prog-5-ussd

A Node.js command-line application simulating an interactive USSD menu (like Mvola), with features such as:

- Balance inquiry  
- Airtime purchase (for self or another number)  
- Money transfer  
- Cash withdrawal  

## 🚀 Quick Start

### Prerequisites

- Node.js v20+  
- npm  
- TypeScript  

### Installation

--- bash
npm install

# Compile TypeScript to JavaScript:
--- bash
npx tsc

# Run the application: 
--- node dist/index.js
